
# FS Manager Elite (FSM 3.0.0) 游戏功能手册

## ⛸️ 1. 核心玩法 (Gameplay Loop)
在 FS Manager Elite 中，你将扮演一名职业花样滑冰选手的经纪人兼教练。你的目标是通过科学的训练、合理的商业运营以及精妙的比赛策略，将一名 14 岁的新秀培养成世界冠军。

## 📊 2. 选手属性与体力系统
选手拥有五项核心实时属性，构成技术 (TEC) 与艺术 (ART) 能力：
- **跳跃 (Jump)**: 影响四周跳等高难度技术动作。
- **旋转 (Spin)**: 提升旋转的稳定与柔韧。
- **步法 (Step)**: 双修技术与艺术的接续步与舞步。
- **表现力 (Perf)**: 增强节目内容分 (PCS) 的感染力。
- **耐力 (Endurance)**: 抗疲劳与体力消耗效率。
- **STA (体力值)**: 核心限制资源。
    - **损耗**: 训练和比赛均会消耗体力。
    - **训练限制**: 若月初体力低于 20%，训练效果会因疲劳大幅衰减。
    - **比赛表现**: 选手的参赛初始体力将直接影响最终得分。体力充沛时表现更稳健，疲劳时则会出现“疲劳减分”。

## 🎭 3. 养成与管理
### 3.1 训练计划
每月可分配 **7个训练槽位** 于跳跃、旋转、步法、表现力、耐力或休息。
- **深度疲劳**: 连续高强度训练会导致体力降至红区，此时必须增加“休息”点数，否则属性将停止成长。

### 3.2 团队与教练
- **教练市场**: 教练对属性成长有不同的加成。

### 3.3 器材系统
- 冰鞋、冰刀、考斯滕均有使用寿命，损毁后属性加成将消失。

## ⚔️ 4. 比赛引擎 2.0
### 4.1 实时评分逻辑
- **TES (技术分)** & **PCS (艺术分)**。
- **体力加权**: 选手的竞技表现受当前体力值修正。
- **策略风险**:
    - **保守发挥**: 极低失误率。
    - **挑战自我**: 追求爆发得分，但失败会导致大幅扣分（Fall Penalty）。

## 🌍 5. 世界排名生态 (ISU World Ranking)
- 150 名 AI 选手动态竞技，遵循 ISU 滚动积分制。
